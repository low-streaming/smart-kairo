# SmartStartScreen for OpenKairo

This integration provides a dynamic launch screen experience for OpenKairo smart home systems distributed via SD cards.

## Features
- **Dynamic News Sensor**: Displays real-time news from openkairo.de on the dashboard.
- **System Status Management**: Tracks local core health.
- **Welcome Experience**: Designed for first-time user onboarding.

## Installation via HACS
1. Add this repository as a **Custom Repository** in HACS (Category: Integration).
2. Install the integration.
3. Restart Home Assistant.
4. Add `smart_start_screen:` to your `configuration.yaml` or set it up via the UI.
